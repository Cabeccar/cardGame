# cardGame
This is a card game

## Programming language
* TypeScript

## Frameworks
* BabylonJs (3.2.0-rc.3)

### Build tool
* Webpack

## Installation
```Shell
$ npm install
```

## Execution
```Shell
$ npm start
```
## Cordova
Cordova Run Android
```Shell
$ npm run cra
```

Cordova Build Android
```Shell
$ npm run cba
```
